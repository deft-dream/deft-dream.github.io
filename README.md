# DARKSEC Blog

安全研究员的攻防记录：SRC漏洞挖掘、CTF靶场实战、红队技术沉淀。

## 特点

- 🎨 赛博朋克风格设计
- ⚡ 纯静态HTML，零依赖
- 🚀 GitHub Pages 自动部署
- 📱 响应式设计，支持移动端

## 本地预览

```bash
# 使用Python
python3 -m http.server 8080

# 使用Node.js
npx serve .

# 使用PHP
php -S localhost:8080
```

## 添加新文章

1. 复制 `templates/article-template.html`
2. 修改内容
3. 保存到 `posts/` 目录
4. 在 `posts/index.html` 和 `index.html` 中添加链接

## 部署

推送到GitHub后，在仓库设置中启用GitHub Pages即可。
